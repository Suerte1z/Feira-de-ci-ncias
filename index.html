Sobre

Hoje você irá conhecer um pouco mais da programação, suas linguagens, e como funciona o desenvolvimento de plataformas.

A programação, ou desenvolvimento, é a arte de desenvolver algum aplicativo, website ou até mesmo um programa, ou jogo.

Programação é o processo de escrita, teste e manutenção de algo para algum dispositivo. O programa é escrito em uma linguagem de programação, embora seja possível, com alguma dificuldade, o escrever diretamente em linguagem de máquina. Diferentes partes de um programa podem ser escritas em diferentes linguagens.

Na prática, existem inúmeras linguagens de programação, que servem para diferentes coisas, e com diferentes níveis de dificuldade. Dentre essas linguagens, as linguagens mais utilizadas atualmente são Python, Java, JavaScript e C++ conheça um pouco mais das linguagens logo abaixo.

Como surgiu?

A programação nos computadores não tem uma data correta de início. Tudo começou na década de 30, com os primeiros computadores elétricos. Em 1948, Konrad Zuse publicou sua criação, a linguagem de programação Plankalkül. Na época, ainda não tinha muita utilidade, então foi esquecida.
